# Personal Notes & Bookmark Manager

## Tech Stack
- Backend: Node.js, Express, MongoDB
- Frontend: Next.js, Tailwind CSS
- Auth: JWT

## Features
- CRUD Notes & Bookmarks
- Search & tag filter
- Auto-fetch bookmark titles
- Dark mode UI
- JWT authentication

## Setup

### Backend
cd backend  
npm install  
npm run dev  

### Frontend
cd frontend  
npm install  
npm run dev  

## API
POST /api/auth/register  
POST /api/auth/login  
POST /api/notes  
GET /api/bookmarks
